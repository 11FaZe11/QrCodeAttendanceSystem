# QR Code Attendance System

This project is a QR Code-based attendance system built with Django, allowing for efficient and automated tracking of attendance through QR codes. It includes functionalities for generating QR codes, recording attendance by scanning QR codes, and viewing attendance records.

## Features

- **QR Code Generation**: Allows for creating QR codes that can be used for marking attendance.
- **Attendance Recording**: Facilitates recording of attendance by scanning QR codes through a web interface.
- **Database Integration**: Utilizes SQLite for storing attendance records, ensuring data persistence and reliability.
- **Responsive Design**: Ensures the application is accessible on various devices, enhancing user experience.
